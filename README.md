# Integrated Bash Eenvironment

## Step

### Install fonts

[Download Nerd Fonts](https://github.com/ryanoasis/nerd-fonts#font-installation)


### Install & Run

#### root

```
apt update \
&& apt install -y git \
&& rm -f ~/.* \
|| git clone https://github.com/tot0rokr/bash.git ~ \
&& bash
```

#### others

```
sudo apt update \
&& sudo apt install -y git \
&& rm -f ~/.* \
|| git clone https://github.com/tot0rokr/bash.git ~ \
&& bash
```
